# Vitest Example

Minimal Vitest + TypeScript project wired for Mutineer.

## Setup

```bash
cd examples/vitest
npm install
```

## Run tests

```bash
npm test
```

## Run Mutineer

Build Mutineer in the repo root, then run the sample:

```bash
cd ../..
npm run build:cli
cd examples/vitest
npm run mutate
```
